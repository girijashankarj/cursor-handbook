# Meta-Prompts for Teams

Meta-prompts are "prompts that generate prompts." Instead of a ready-to-paste prompt, a meta-prompt asks the AI to generate a customized prompt, document, or workflow tailored to your specific project.

## When to use meta-prompts

- You want output customized to your team's tech stack, conventions, and codebase
- You need to create reusable team resources (onboarding docs, review checklists, runbooks)
- You want to scale a workflow across multiple projects with different stacks
- You're setting up cursor-handbook for a new team and want tailored guidance

## How to use

1. Open the meta-prompt file for your goal
2. Paste it into Cursor chat
3. The AI reads your project context (project.json, codebase, rules) and generates a tailored output
4. Save the output as a team resource (wiki, repo doc, shared prompt)
